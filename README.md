# King of Iron Fist — Login Page UI

A styled login/register page UI built with HTML,CSS and Javascript

## Features

- Login and register forms with tab switching
- Responsive design (desktop + mobile)
- Custom checkbox ("Remember me") and "Forgot password?" link
- Animated gradient borders and 3D hover effect
- Blurred background image with dark overlay
- Brand-colored purple theme with neon accent links
- Font Awesome icons and custom PNG icons

## Usage

Open `index.html` in any modern browser.

## Structure

```
├── index.html          — Main page
├── style.css           — All styles
├── script.js           — Form switching logic
├── background.jpg      — Blurred page background
├── Kazuya-poster.jpg   — Right-panel background
├── icon1.png           — Tech icon
├── icon2.png           — Tech icon
├── favicon.png         — Page favicon
└── README.md
```

## Customization

Colors are controlled via CSS custom properties in `:root` inside `style.css`:

```css
--primary: #833996;
--secondary: #4803a0;
--dark: #0a0a14;
--darker: #05050c;
--light: rgba(255, 255, 255, 0.9);
```
